# Linux Privilege Escalation Lab

## Objective
Escalate privileges from a limited user account to root access.

## Tools
- Kali Linux
- Linux commands

## Method

1. Enumerate system information
2. Identify misconfigurations
3. Exploit privilege escalation path

## Result
Root privileges were obtained in the lab environment.

## Security Lesson
Proper user permission management and patching reduce privilege escalation risks.
