# Cybersecurity Labs Portfolio

This repository documents hands-on cybersecurity labs performed during my BSc Cyber Security program at Nile University of Nigeria and through independent practice.

The labs demonstrate practical experience with penetration testing, vulnerability assessment, and network security tools.

## Tools Used

- Kali Linux
- Nmap
- Netcat
- Nessus
- OpenVAS
- OWASP ZAP
- Linux

## Lab Projects

| Lab | Description |
|----|----|
| Nmap Network Scanning | Network reconnaissance and port scanning |
| Netcat Banner Grabbing | Service enumeration and banner grabbing |
| Nessus Vulnerability Scan | Identifying system vulnerabilities |
| Linux Privilege Escalation | Escalating privileges in a controlled lab |

## Objective

These labs demonstrate the penetration testing workflow:

1. Reconnaissance  
2. Vulnerability discovery  
3. Exploitation  
4. Privilege escalation  
5. Security mitigation
